# AI Lecture Assistant

A modular Streamlit-based application for turning lecture videos into searchable, AI-generated learning materials.

## Structure

- app/: Streamlit UI and reusable components
- backend/: processing pipeline, services, generators, prompts, database access, and helpers
- data/: uploaded media, audio, transcripts, vector storage, and exports
- config/: centralized settings and constants
- tests/: automated tests

## Getting started

1. Create and activate a Python environment.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python run.py`
